==========================
Aprenda Python programando
==========================

Este conjunto de tutoriais consiste em uma primeira introdução ao Python para
pessoas que possuam alguma (mas não necessariamente muita) experiência de 
programação em outras linguagens. Cada arquivo possui um pequeno exemplo e 
explicação de um conceito básico e normalmente um ou dois exercícios. Leia as
explicações e tente fazer os exercícios. Quando acabar, você vai saber se virar
muito bem no Python!
